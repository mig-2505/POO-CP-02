# CheckPoint 02 - Programação Orientanda a Objeto
Projeto desenvolvido em Java para praticar conceitos de Programação Orientada a Objetos (POO).

## Sobre
O sistema simula um cenário de transporte de pacotes, utilizando classes e regras básicas.

## Conceitos utilizados
1. Classes e Objetos
2. Encapsulamento
3. Construtores
4. Getters e Setters
5. Relacionamentos
6. Herança
